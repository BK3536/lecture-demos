Teaching-model note
This demo uses a scaled discrete baseband model instead of orbital geometry, so the browser can render the SAR chain cleanly and reproducibly. Each point target is blurred by a range chirp and an azimuth chirp, then recovered with matched filters.

What to look for:
  1. Raw echo is unfocused in both dimensions.
  2. After range compression sharpens only the range axis.
  3. Focused SAR image sharpens both range and azimuth.
  4. PSF shows the isolated point response for the same settings.
